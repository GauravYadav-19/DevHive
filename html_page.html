HTML Tutorial for Beginners

HTML (HyperText Markup Language) is the standard markup language for creating web pages. It describes the structure of a webpage using elements represented by tags like <h1>, <p>, <a>, etc.

🌐 Basic Structure of an HTML Document

Every HTML page starts with a basic structure. Here's the template:


<!DOCTYPE html>
<html> 
  <head>
    <title>Page Title</title>
  </head>
  <body>
    <h1>This is a heading</h1>
    <p>This is a paragraph.</p>
  </body>
</html>

🧱 Headings and Paragraphs

HTML provides 6 heading levels from <h1> (largest) to <h6> (smallest).


<h1>Main Heading</h1>
<h2>Subheading</h2>
<p>This is a paragraph describing something important.</p>

🔗 Links and 🖼️ Images

Use the <a> tag to link to pages and <img> to display images.


<a href="https://devhive.com">Visit DevHive</a>

<img src="https://via.placeholder.com/150" alt="Placeholder Image">

📋 Lists in HTML

There are two types of lists:

  • <ul> – Unordered List
  • <ol> – Ordered List

<ul>
  <li>HTML</li>
  <li>CSS</li>
</ul>

<ol>
  <li>Install VS Code</li>
  <li>Write your first HTML page</li>
</ol>

📊 Tables

Tables are used to display data in rows and columns.


<table>
  <tr>
    <th>Name</th>
    <th>Age</th>
  </tr>
  <tr>
    <td>Gaurav</td>
    <td>21</td>
  </tr>
</table>

📝 HTML Forms

Forms are used to collect user input.


<form>
  Name: <input type="text" name="name"><br>
  Email: <input type="email" name="email"><br>
  <input type="submit" value="Submit">
</form>

📌 Semantic Elements

Semantic tags like <header>, <nav>, <section>, and <footer> describe the meaning of content.


<header>My Site Header</header>
<nav>Navigation Links</nav>
<section>Main content</section>
<footer>© 2025 DevHive</footer>
