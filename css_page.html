What is CSS?

CSS (Cascading Style Sheets) is used to style and layout web pages. It controls the colors, fonts, spacing, positioning, and even animations of elements on a webpage. 


/* Example */
body {
  background-color: lightblue;
  font-family: Arial, sans-serif;
}

CSS Syntax

A CSS rule is made of a selector and a declaration block.


selector {
  property: value;
}

Example:


h1 {
  color: red;
  text-align: center;
}

  • Internal CSS â€“ inside a <style> tag in the HTML head

CSS Selectors

Selectors are used to "select" HTML elements you want to style.

  • Element Selector: p { color: red; }
  • ID Selector: #title { font-size: 30px; }
  • Class Selector: .highlight { background: yellow; }

Colors & Backgrounds

  background-color: #f0f0f0;
}

CSS Box Model

Every element in CSS is a box, consisting of:

  • Content
  • Padding
  • Border
  • Margin

div {
  width: 200px;
  padding: 20px;
  border: 2px solid black;
  margin: 10px;
}

Responsive Design

Responsive CSS makes your website look good on all devices.


@media (max-width: 600px) {
  body {
    background-color: lightcoral;
  }
}
