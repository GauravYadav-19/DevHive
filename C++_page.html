Introduction to C++

C++ is a general-purpose, high-performance programming language created by Bjarne Stroustrup as an extension of the C programming language. It supports both procedural and object-oriented programming, making it one of the most powerful and widely used languages.

Basic Structure of a C++ Program


#include <iostream>
using namespace std;

int main() {
    cout << "Hello, World!" << endl;
    return 0;
}

Variables and Data Types

Variables store data values. C++ has many built-in types:

  • int → integer (e.g., 5, -10)
  • float → decimal (e.g., 3.14)
  • double → double-precision decimal
  • char → single character ('A')
  • string → text ("Hello")
  • bool → true/false

#include <iostream>
using namespace std;

int main() {
    int age = 21;
    float pi = 3.14;
    char grade = 'A';
    string name = "Gaurav";
    bool isStudent = true;

    cout << "Name: " << name << ", Age: " << age << endl;
    return 0;
}

Operators in C++

C++ has arithmetic, relational, logical, and assignment operators:

  • Arithmetic: +, -, *, /, %
  • Relational: ==, !=, <, >, <=, >=
  • Logical: &&, ||, !
  • Assignment: =, +=, -=, *=, /=

int a = 10, b = 5;
cout << (a + b) << endl; // 15
cout << (a > b) << endl;  // 1 (true)

Control Flow (if, switch, loops)

if-else


int marks = 85;
if(marks >= 90) cout << "Grade A";
else if(marks >= 75) cout << "Grade B";
else cout << "Grade C";

switch


int day = 3;
switch(day) {
    case 1: cout << "Monday"; break;
    case 2: cout << "Tuesday"; break;
    default: cout << "Other Day";
}

Loops


// For loop
for(int i=1; i<=5; i++) cout << i << " ";

// While loop
int j = 1;
while(j <= 5) { cout << j << " "; j++; }

// Do-while loop
int k = 1;
do { cout << k << " "; k++; } while(k <= 5);

Functions in C++


#include <iostream>
using namespace std;

int add(int a, int b) {
    return a + b;
}

int main() {
    cout << "Sum: " << add(5, 10);

}

Arrays and Strings


// Array Example
int arr[5] = {1,2,3,4,5};
for(int i=0; i<5; i++) cout << arr[i] << " ";

// String Example
string s = "DevHive";

Pointers in C++


int x = 10;
int* ptr = &x;

cout << "Value of x: " << x << endl;
cout << "Address of x: " << &x << endl;
cout << "Pointer ptr stores: " << ptr << endl;
cout << "Value via ptr: " << *ptr << endl;

Object-Oriented Programming (OOP)


#include <iostream>
using namespace std;

class Student {
public:
    string name;
    int age;

    void display() {
        cout << "Name: " << name << ", Age: " << age;
    }
};

int main() {
    Student s1;
    s1.name = "Gaurav";
    s1.age = 21;
    s1.display();
    return 0;
}

Standard Template Library (STL)

STL provides pre-built classes and functions for common data structures. 


#include <iostream>
#include <vector>
using namespace std;

int main() {
    vector<int> v = {1,2,3};
    v.push_back(4);

    for(int x : v) cout << x << " ";  // 1 2 3 4
    return 0;
}

File Handling in C++


#include <iostream>
#include <fstream>
using namespace std;

int main() {
    ofstream fout("sample.txt");
    fout << "Hello, File!";
    fout.close();

    ifstream fin("sample.txt");
    string text;
    getline(fin, text);
    cout << text;
    fin.close();

    return 0;
}

Conclusion

C++ is a versatile language that blends low-level efficiency with high-level abstractions. From system programming to AI, competitive coding, and game development, C++ remains one of the most valuable programming skills to learn.
