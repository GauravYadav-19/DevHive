Introduction

JavaScript is a powerful programming language that adds interactivity, animations, and logic to web pages. While HTML structures the content and CSS styles it, JavaScript brings your page to life.

Why Learn JavaScript?

  • It’s the most popular programming language in the world.
  • Essential for web development (frontend + backend).
  • Used in modern frameworks like React, Angular, and Vue.
  • Also used outside browsers (Node.js, mobile apps, etc.).

Basic Syntax


// JavaScript is written inside <script> tags or separate .js files
console.log("Hello, World!");

Variables

Variables store data values. JavaScript has three main ways to declare them:


var name = "DevHive";   // Old way
let age = 21;           // Modern, block-scoped
const pi = 3.14159;     // Constant (cannot change)

Data Types

  • String → "Hello"
  • Number → 42
  • Boolean → true / false
  • Array → [1, 2, 3]
  • Object → {name: "Gaurav", age: 21}
  • Null, Undefined

Functions

Functions allow you to reuse code.


function greet(name) {
  return "Hello " + name;
}

console.log(greet("DevHive"));

Events

JavaScript responds to user actions (click, hover, input, etc.).

DOM Manipulation

JavaScript can modify HTML/CSS dynamically.


document.getElementById("demo").innerHTML = "Text changed!";
